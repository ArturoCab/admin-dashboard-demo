# Admin Dashboard Demo

A responsive admin dashboard layout built with HTML and CSS.

This project focuses on layout composition using CSS Grid and Flexbox, 
card-based UI design, reusable components, and modern styling techniques.

## Features

- CSS Grid for main layout structure
- Flexbox for internal alignment
- Reusable card components
- Sidebar navigation
- Header with search bar and actions
- Hover interactions and subtle animations
- Pseudo-elements for visual accents

## Technologies Used

- HTML5
- CSS3 (Grid, Flexbox, Custom Properties)

## What I Practiced

- Combining Grid and Flexbox effectively
- Structuring scalable layouts
- Creating modern UI elements without frameworks
- Improving visual hierarchy and spacing
